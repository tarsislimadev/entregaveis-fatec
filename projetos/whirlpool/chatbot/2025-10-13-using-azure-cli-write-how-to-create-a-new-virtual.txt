> using azure cli, write how to create a new virtual machine

● Here's how to create a new virtual machine using Azure CLI:

  # Create a resource group (if you don't have one)
  az group create --name myResourceGroup --location eastus

  # Create a virtual machine
  az vm create \
    --resource-group myResourceGroup \
    --name myVM \
    --image Ubuntu2204 \
    --admin-username azureuser \
    --generate-ssh-keys \
    --size Standard_B1s \
    --location eastus2

  Key parameters:
  - --resource-group: Resource group name
  - --name: VM name
  - --image: OS image (Ubuntu2204, Win2022Datacenter, etc.)
  - --admin-username: Admin user name
  - --generate-ssh-keys: Auto-generate SSH keys for Linux VMs
  - --size: VM size (Standard_B1s is basic, Standard_D2s_v3 for more power)
  - --location: eastus (East US 2 Region)

 For Windows VMs, replace --generate-ssh-keys with --admin-password and provide a password when prompted.
