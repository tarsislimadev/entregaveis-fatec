## Equações de 1o e 2o graus

### Equação de 1o grau

Uma equação de 1º grau possui a forma geral ax + b = 0, onde a ≠ 0.

**Exemplo:**
2x + 4 = 0

**Resolução:**
2x = -4
x = -4/2
x = -2

Ou seja, x = -2 é a solução.

### Equação de 2o grau

Uma equação de 2º grau possui a forma geral ax² + bx + c = 0, onde a ≠ 0.

A solução é dada pela fórmula de Bhaskara:
x = [-b ± √(b² - 4ac)] / (2a)

**Exemplo:**
x² - 5x + 6 = 0

**Resolução:**
a = 1, b = -5, c = 6
Δ = (-5)² - 4*1*6 = 25 - 24 = 1
x₁ = [5 + 1]/2 = 3
x₂ = [5 - 1]/2 = 2

As soluções são x = 3 e x = 2.
