# Atividade 02: Classificação de Notas

Uma escola deseja automatizar a classificação dos alunos com base em suas notas finais

O sistema deve seguir as seguintes regras:
- Nota maior ou igual a 90: Exibir "Aprovado com Excelência" 
- Nota entre 70 e 89: Exibir "Aprovado" 
- Nota entre 50 e 69: Exibir "Recuperação" 
- Nota menor que 50: Exibir "Reprovado" 

# Tarefa

Crie um programa em Python que:

- Peça ao usuário para digitar a nota do aluno (um número entre 0 e 100)
- Utilize a estrutura if para classificar a nota conforme as regras acima
- Exiba uma mensagem informando a situação do aluno.

Salve o programa com o nome [atividade_aula_02_02.py](./atividade_02.py)
