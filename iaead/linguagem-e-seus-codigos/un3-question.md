# Linguagem e seus Códigos

## Unidade 3

### Perguntas avaliativas

Pergunta: Um sistema de PLN deve ser capaz de reconhecer que "feliz", "felicidade" e "infeliz" compartilham um mesmo campo semântico. Qual elemento morfológico contribui para essa relação entre palavras?

Resposta: Radical ou raiz comum

Pergunta: Em linguística computacional, identificar palavras compostas é fundamental para preservar significados. Dentre os exemplos abaixo, qual representa corretamente uma palavra composta?

Resposta: Guarda-chuva

Pergunta: Durante a análise de dados de uma plataforma educacional, percebeu-se que a variação morfológica de termos-chave dificultava a categorização dos conteúdos. Uma forma de reduzir essa variação para melhorar a categorização é:

Resposta: Aplicar stemming ou lematização aos termos

Pergunta: Durante a criação de um modelo de classificação de textos, um aluno questiona se vale a pena remover palavras como "a", "de", "e". Essas palavras são geralmente removidas por pertencerem a uma classe gramatical com baixo valor semântico. Qual é essa classe?

Resposta: Conjunções e preposições
