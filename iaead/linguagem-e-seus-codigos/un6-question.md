# Linguagem e seus Códigos

## Unidade 2

### Perguntas Avaliativas

Pergunta: Em uma conversa com um assistente virtual, o usuário pergunta: “Você sabe que horas são?” O sistema responde: “Sim.”. O usuário considera a resposta inadequada. Esse erro mostra que o sistema falhou em reconhecer qual aspecto da linguagem?

Resposta: A intencionalidade pragmática da pergunta.

Pergunta: Ao preparar um conjunto de dados para treinar um modelo de classificação de e-mails, um cientista de dados aplica as etapas de: remoção de pontuação, tokenização, conversão para minúsculas e eliminação de palavras comuns como “de”, “o”, “em”. Essas etapas fazem parte de que processo no tratamento de linguagem natural?

Resposta: Pré-processamento textual.

Pergunta: Um time de cientistas de dados desenvolve um sistema de sumarização automática de textos. Para isso, utilizam uma base linguística que considera as estruturas sintáticas mais informativas nas orações. O uso da teoria linguística nesse processo indica a importância de:

Resposta: Base teórica da linguística na modelagem computacional.

Pergunta: Ao analisar a frase “O cientista analisou os dados com cuidado”, um aluno observou que é possível alterar a estrutura da frase sem perder o sentido, trocando palavras por outras equivalentes, como “O pesquisador examinou os registros com atenção”. Essa análise evidencia que o aluno está reconhecendo:

Resposta: A relação paradigmática de substituição entre os termos.
