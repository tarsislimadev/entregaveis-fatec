# Linguagem e seus Códigos

## Unidade 2

### Exercícios de Fixação

Pergunta: Em um projeto de análise de dados textuais, uma equipe de cientistas de dados precisa compreender como a linguística pode auxiliar na organização e interpretação de textos. Qual é a principal contribuição dos fundamentos da linguística para a Ciência de Dados?

Resposta: Permitir a análise estruturada da linguagem natural para extração de informações.

Pergunta: Um cientista de dados precisa programar um sistema de análise semântica que compreenda as relações entre palavras. De acordo com Saussure, o signo linguístico é formado por:

Resposta: Significado e significante.
