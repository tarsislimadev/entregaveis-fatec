# Gestão de Projetos

Pergunta: A Lean Inception é uma metodologia colaborativa para definição do MVP (Produto Viável Mínimo) que alinha times e stakeholders. Sobre a Lean Inception, quais são as afirmações corretas? I. A Lean Inception utiliza dinâmicas colaborativas para alinhar objetivos do projeto. II. O principal objetivo da Lean Inception é criar um backlog detalhado. III. O MVP deve ser validado o mais cedo possível com o cliente.

Resposta: I e III estão corretas.

Pergunta: A Lean Inception é um processo curto e intensivo voltado para a definição de prioridades e objetivos iniciais do produto. Qual é o principal resultado esperado ao final de uma Lean Inception?

Resposta: A definição do Produto Viável Mínimo (MVP).

Pergunta: Durante uma Lean Inception, o facilitador percebe que alguns participantes estão falando demais e não permitindo que outros contribuam. O facilitador precisa garantir que todos participem igualmente para que o processo de construção do MVP seja colaborativo. Qual seria a ação mais apropriada para o facilitador tomar nesta situação?

Resposta: Interromper a pessoa e dizer que ela está tomando muito tempo.

Pergunta: Uma empresa de saúde está realizando uma Lean Inception para desenvolver uma nova plataforma de agendamento de consultas. No início do workshop, algumas partes interessadas têm ideias divergentes sobre a visão do produto. Você, como facilitador, propõe uma atividade para alinhar essas ideias. Qual atividade seria mais apropriada no início do workshop para alinhar a visão do produto?

Resposta: Realizar a atividade de "objetivos do produto".

Pergunta: Uma equipe de desenvolvimento de software está participando de uma Lean Inception para criar um novo sistema de gestão financeira. O CEO insiste que todas as funcionalidades propostas pela equipe são "essenciais" para o produto. Como facilitador da Lean Inception, você precisa ajudar a equipe a definir o Produto Mínimo Viável (MVP). O que é mais importante ao definir o MVP durante a Lean Inception?

Resposta: Priorizar apenas as funcionalidades críticas que validam a proposta de valor do produto
