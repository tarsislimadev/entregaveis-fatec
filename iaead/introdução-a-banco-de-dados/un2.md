Resposta: Esquema

Pergunta: A arquitetura de três esquemas é uma ferramenta adequada para visualizar os níveis do esquema em um sistema de banco de dados. Analise o exemplo a seguir: O gerente de uma agência bancária tem permissão para ver as operações e outros dados dos seus clientes, porém, isso não é permitido para outros gerentes que não sejam os responsáveis por essas contas. Isso acontece devido a características oferecidas pela arquitetura de três esquemas em um banco de dados.

Resposta: Suporte a múltiplas visões (views) de usuários.

Pergunta: Cada vez que um registro for inserido, alterado ou apagado o estado do banco de dados será modificado.

Resposta: Instâncias do banco de dados, pois as operações modificam o conteúdo armazenado

Pergunta: Leia com atenção as afirmativas sobre Projeto de banco de dados: I. É um processo simples que normalmente precisa de algumas decisões em diferentes níveis, o que é habitual para uma equipe experiente. II. Modelagem conceitual é uma das fases do projeto na qual é construído um modelo conceitual, na forma de um diagrama entidade-relacionamento. Esse modelo captura as necessidades da organização em termos de armazenamento de dados de forma independente de implementação. III. Na fase do Projeto lógico, o objetivo é transformar o modelo conceitual obtido na primeira fase em um modelo lógico que define como o banco de dados será implementado em um SGBD específico.

Resposta: As afirmativas II e III são verdadeiras.
