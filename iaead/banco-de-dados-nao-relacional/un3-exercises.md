# Banco de Dados Não Relacional

## Unidade 3

### Exercícios de Fixação

Pergunta: Bancos de Dados NoSQL se popularizaram como solução para lidar com dados em larga escala, estruturados ou não, com foco em desempenho, flexibilidade e alta disponibilidade. Entre os diversos modelos, destacam-se os orientados a documentos, chave-valor, grafos e colunares. Sobre as características dos Bancos de Dados NoSQL orientados a documentos, assinale a alternativa correta. _____ $CATEGORY: top/BD-NÃO RELACIONAL

Resposta: Permitem armazenar dados sem estrutura fixa em documentos JSON ou BSON, com suporte a campos aninhados e diferentes estruturas entre registros.

Pergunta: Com o crescimento exponencial do volume de dados gerados por aplicações modernas e o aumento da diversidade de formatos, soluções como Data Warehouse e Data Lake se tornaram indispensáveis para a análise e a gestão eficiente das informações nas organizações. Assinale a alternativa que apresenta corretamente uma diferença fundamental entre Data Warehouse e Data Lake.

Resposta: Data Warehouse exige estruturação prévia dos dados (schema-on-write), enquanto o Data Lake armazena dados brutos para posterior processamento (schema-on-read).
