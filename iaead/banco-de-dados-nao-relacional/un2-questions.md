# Banco de Dados Não Relacional

## Unidade 2

### Perguntas Avaliativas

Pergunta: Com a evolução tecnológica, surgiram modelos híbridos que combinam as qualidades de diferentes repositórios de dados. O que define a arquitetura conhecida como “Lakehouse”?

Resposta: Combina a flexibilidade do Data Lake com a estrutura e governança do Data Warehouse.

Pergunta: A integração de Data Lake com ferramentas de IA e machine learning ampliou seu potencial analítico. Qual vantagem a integração de IA e machine learning proporciona aos Data Lakes?

Resposta: Permite treinar e aplicar modelos diretamente sobre dados brutos em grande escala.

Pergunta: Organizações modernas utilizam diferentes estratégias de armazenamento de dados para lidar com os desafios de volume, variedade e velocidade. Qual a principal diferença entre os modelos de dados adotados por Data Warehouse e Data Lake?

Resposta: O Data Warehouse exige esquema pré-definido (schema-on-write), enquanto o Data Lake aplica o esquema no momento da leitura (schema-on-read).

Pergunta: O termo "data swamp" é frequentemente utilizado como um alerta em relação ao uso inadequado dos Data Lakes. O que caracteriza um “data swamp” no contexto de um Data Lake?

Resposta: Repositório desorganizado e inutilizável, causado por falta de controle, qualidade e governança dos dados.
