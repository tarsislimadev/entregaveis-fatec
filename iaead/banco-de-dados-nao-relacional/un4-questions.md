# Banco de Dados Não Relacional

## Unidade 3

### Perguntas Avaliativas

Pergunta: Um banco de dados colunar, como o Apache Cassandra, utiliza uma chave primária composta por uma partição e uma chave de ordenação. Isso influencia diretamente como os dados são organizados e acessados. Em uma tabela do Cassandra, qual a função da chave de partição e da chave de clustering, respectivamente?

Resposta: Distribuir os dados entre os nós do cluster e ordenar os registros dentro da partição.

Pergunta: Na escolha de um modelo NoSQL, é necessário avaliar os critérios de flexibilidade, suporte a relacionamentos e tipo de consulta desejada. Em relação ao suporte a relacionamentos entre dados, qual modelo NoSQL oferece a melhor solução?

Resposta: Modelo de grafos, pois é especializado em representar e navegar entre conexões entre entidades.

Pergunta: O MongoDB é amplamente utilizado em aplicações web modernas por sua estrutura flexível e integração com objetos JSON. Qual característica do MongoDB favorece sua integração com linguagens de programação modernas?

Resposta: Utilização de documentos BSON, cuja estrutura se assemelha a objetos JSON usados em linguagens como JavaScript e Python

Pergunta: Apesar de ser eficiente em consultas relacionais, o modelo de grafos possui limitações técnicas que precisam ser consideradas no planejamento de sistemas em grande escala. Qual das alternativas apresenta corretamente uma limitação do modelo de grafos?

Resposta: Dificuldade de escalabilidade em redes muito densas e altamente conectadas.
