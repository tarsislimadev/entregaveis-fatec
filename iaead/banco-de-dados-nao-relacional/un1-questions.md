# Banco da Dados Não Relacional

## Unidade 1

### Perguntas Avaliativas

Pergunta: Ao tratar de dados organizados, o termo “estruturado” é frequentemente empregado para descrever formatos que seguem modelos formais e rígidos. Sobre dados estruturados, é correto afirmar que:

Resposta: São organizados segundo um esquema fixo e armazenados em Bancos Relacionais com suporte a SQL.

Pergunta: O MongoDB é um dos Bancos de Dados NoSQL mais populares, especialmente em projetos web e em nuvem. Sobre o MongoDB, marque a alternativa correta:

Resposta: É um Banco de Dados orientado a documentos, que armazena dados no formato BSON, oferecendo alta flexibilidade de esquema.

Pergunta: A distinção entre dado e informação é essencial para compreender como o conhecimento é gerado a partir do tratamento dos dados coletados. Qual das opções a seguir melhor diferencia dados de informação?

Resposta: Dado é um elemento bruto; informação é o resultado da organização e interpretação desse dado em um contexto.

Pergunta: As arquiteturas de dados modernas requerem mecanismos de armazenamento compatíveis com os 3Vs do Big Data: volume, velocidade e variedade. Qual das alternativas a seguir melhor representa uma vantagem dos Bancos de Dados NoSQL frente aos relacionais, no contexto dos 3Vs?

Resposta: A capacidade de escalar horizontalmente, distribuindo dados entre múltiplos servidores.
