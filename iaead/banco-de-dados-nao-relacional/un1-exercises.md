# Banco de Dados Não Relacional

Pergunta: Dentre os diversos modelos de dados oferecidos pelos Bancos NoSQL, os bancos orientados a grafos são utilizados para modelar relacionamentos complexos entre entidades, oferecendo alta performance em consultas relacionais complexas. Qual característica melhor define um banco de dados orientado a grafos?

Resposta: Armazena entidades como nós e seus relacionamentos como arestas, permitindo a análise de conexões complexas.

Pergunta: O crescimento exponencial na geração de dados e a diversificação de seus formatos impulsionaram o surgimento de novas soluções para armazenamento e gerenciamento. Entre elas, os Bancos de Dados NoSQL se destacam por sua escalabilidade, flexibilidade de esquema e capacidade de lidar com grandes volumes de dados não estruturados. Qual das alternativas a seguir expressa corretamente uma vantagem fundamental dos Bancos de Dados NoSQL, em comparação aos Bancos de Dados Relacionais?

Resposta: São escaláveis horizontalmente, permitindo a distribuição de dados entre vários servidores, o que facilita o processamento de grandes volumes de dados.
