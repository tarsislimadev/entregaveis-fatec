# Redes de Computadores

## Unidade 3

### Perguntas Avaliativas

Pergunta: Um técnico de suporte deseja verificar se um computador consegue alcançar outro na rede, usando comandos simples do sistema operacional que testam conectividade. Qual protocolo pode ser usado para diagnóstico de conectividade por meio do comando "ping"?

Resposta: ICMP.

Pergunta: Uma universidade planeja conectar diferentes blocos de seu campus por meio de uma rede de alto desempenho, que ofereça alta taxa de transmissão e seja resistente a interferências eletromagnéticas do ambiente urbano. Qual meio de transmissão é mais adequado?

Resposta: Fibra óptica monomodo.

Pergunta: Uma plataforma de vídeo sob demanda quer garantir que os usuários possam assistir aos conteúdos com o menor atraso possível, mesmo que algumas perdas de dados ocorram ocasionalmente sem comprometer a experiência. Qual protocolo de transporte é mais adequado?

Resposta: UDP.

Pergunta: Ao digitar um endereço de site no navegador, um aluno se questiona como o computador descobre qual o endereço IP correspondente para acessar o conteúdo desejado. Qual protocolo da Camada de Aplicação traduz esse nome para um endereço IP?

Resposta: DNS.
