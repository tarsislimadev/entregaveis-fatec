# Redes de Computadores

## Unidade 4

### Exercícios de Fixação

Pergunta: Um projeto de cabeamento estruturado está sendo implementado em um campus universitário com vários blocos. O que a norma ABNT NBR 14565 define quanto à instalação da infraestrutura?

Resposta: Roteamento adequado, separação entre cabos de energia e dados e distâncias máximas permitidas.

Pergunta: Um engenheiro de redes está projetando a infraestrutura de um datacenter de médio porte. Ele precisa seguir normas específicas para garantir segurança e confiabilidade dos serviços oferecidos. Qual norma técnica é mais indicada para guiar o projeto de cabeamento estruturado em datacenters?

Resposta: TIA-942, que define padrões específicos para infraestrutura de datacenters.

Pergunta: Um técnico de redes recebeu a tarefa de instalar uma rede em uma indústria com ambiente eletromagneticamente ruidoso e precisa seguir padrões apropriados. Qual norma técnica internacional oferece diretrizes para ambientes industriais com alto nível de interferência?

Resposta: ISO/IEC 24702, que trata do cabeamento estruturado em ambientes industriais.

Pergunta: Um hospital está implantando uma nova infraestrutura de rede para monitoramento em tempo real de pacientes. Qual norma deve ser consultada para garantir conformidade com requisitos desse tipo de ambiente?

Resposta: ISO/IEC 27001, para gerenciamento de serviços de TI.
