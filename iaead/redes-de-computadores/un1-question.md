# Redes de Computadores

## Unidade 1

### Perguntas avaliativas

Pergunta: Em uma empresa de streaming, a comunicação com os servidores precisa ser confiável e de baixa latência. Qual protocolo é mais adequado para garantir a entrega correta dos dados?

Resposta: TCP

Pergunta: Um engenheiro precisa estabelecer uma comunicação entre dispositivos, via fibra óptica, em uma rede industrial. Qual a principal vantagem do uso de fibra óptica nesse caso?

Resposta: Alta velocidade e imunidade a interferências eletromagnéticas.

Pergunta: Durante uma aula, um professor menciona que todos os dispositivos de sua rede estão conectados a um switch central. Qual é a topologia física utilizada nessa rede?

Resposta: Estrela.

Pergunta: Um administrador de rede deseja atualizar uma estrutura física simples que está utilizando apenas um cabo para conexão de múltiplos pontos de rede. A partir dessa característica, qual topologia ele vai utilizar para fazer essa atualização?

Resposta: Barramento
